# PRODIGY_WD_01 - Interactive Navigation Menu

## Overview
This project is a modern and interactive navigation menu created using HTML, CSS, and JavaScript. The navigation bar remains fixed at the top of the page and changes style when scrolling or hovering over menu items.

## Features
- Fixed navigation bar
- Smooth scrolling navigation
- Hover effects on menu items
- Navbar background color changes on scroll
- Responsive design
- Hero section with background image
- About, Services, Projects, and Contact sections
- Modern UI with cards and animations

## Technologies Used
- HTML5
- CSS3
- JavaScript
